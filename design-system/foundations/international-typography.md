# International Typography

## Japanese
- Pair Latin + JP intentionally; avoid default system-only stacks for brand work.
- Line length and leading differ from Latin; test 日本語 body copy.
- Respect 禁則 and `palt` judiciously.

## RTL
- Mirror layout, not meaning-bearing icons blindly.
- Logical CSS properties (`margin-inline`) over left/right.
